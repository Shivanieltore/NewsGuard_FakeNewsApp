NewsGuard: Detecting the Lies Behind the Headlines

This is a simple machine learning web application that detects fake news using NLP techniques.
It uses Naive Bayes for classification, TF-IDF for vectorization, and Flask for the interface.

Steps to Run:
1. Install required packages using: pip install -r requirements.txt
2. Run the Flask app using: python app.py
3. Open the browser at http://127.0.0.1:5000/
